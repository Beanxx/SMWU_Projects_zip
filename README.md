### 작동 과정
1. 라즈베리파이의 터미널 창에 학번과 이름을 입력한다.
2. 아두이노에 장착된 적외선 온도 센서에 손목을 가까이 가져다댄다.
3. 아두이노가 측정한 온도 정보를 라즈베리파이로 보낸다(아두이노와 라즈베리파이는 USB 케이블로 연결됨)
4. 사용자 학번, 이름, 체온을 터미널 창에 띄우고 .txt 파일에 저장한다.
